# Swagger + OpenAPI Study

Projeto de estudo de documentação de APIs usando **Swagger UI** e **OpenAPI 3.0** com **NestJS + TypeScript**.

## Stack

- NestJS 11
- TypeScript
- @nestjs/swagger (Swagger UI + geração OpenAPI)

## Como rodar

```bash
npm install
npm run start:dev
```

## Exemplo Spec-First

O arquivo `openapi-spec-example.yaml` contém a mesma API escrita manualmente no formato OpenAPI 3.0 (abordagem Spec-First), para comparação com a abordagem Code-First usada neste projeto.
